### Installation

Add this to your module `build.gradle` file:
```gradle
dependencies {
	...
	 compile 'com.github.tarek360.RichPathAnimator:richpath:0.0.1'
}
```

Add this to your root `build.gradle` file (**not** your module `build.gradle` file) :
```gradle
allprojects {
	repositories {
		...
		maven { url "https://jitpack.io" }
	}
}
```
